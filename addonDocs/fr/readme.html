Padronix - Formateur d'Articles Académiques

Version : 2026.1.0

Description

Padronix est un module complémentaire pour NVDA qui aide à structurer et formater des articles académiques. Avec l'intégration de modèles d'IA, il permet de générer des documents aux normes ABNT, APA, IEEE et autres formats académiques de manière accessible et automatisée.

Installation

  1. Téléchargez le fichier .nvda-addon
  2. Ouvrez NVDA → Outils → Gérer les modules complémentaires
  3. Cliquez sur "Installer..." et sélectionnez le fichier
  4. Redémarrez NVDA lorsque vous y êtes invité

Utilisation

  1. Appuyez sur NVDA+Ctrl+Shift+F pour ouvrir Padronix
  2. Chargez un fichier TXT avec le texte de l'article
  3. Importez un fichier JSON structuré (optionnel)
  4. Cliquez sur "Générer le document DOCX"
  5. Le fichier sera enregistré dans le dossier Documents

Paramètres

Accès : NVDA → Préférences → Paramètres → Padronix

  • Fournisseur d'IA (Google, OpenAI, OpenRouter, Ollama)
  • Clé API
  • Modèle
  • Norme académique (ABNT, APA, IEEE, etc.)

Prérequis

  • NVDA 2024.1 ou supérieur
  • Python 3.11+ (inclus dans NVDA 2026+)
  • Connexion Internet (sauf pour Ollama local)
  • Bibliothèques : python-docx, lxml (incluses)
