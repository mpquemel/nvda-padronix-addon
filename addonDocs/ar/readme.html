Padronix - منسق المقالات الأكاديمية

الإصدار: 2026.1.0

الوصف

Padronix هو إضافة لـ NVDA تساعد في هيكلة وتنسيق المقالات الأكاديمية. من خلال التكامل مع نماذج الذكاء الاصطناعي، يتيح إنشاء مستندات وفق معايير ABNT وAPA وIEEE وغيرها من الصيغ الأكاديمية بطريقة سهلة وآلية.

التثبيت

  1. قم بتنزيل ملف .nvda-addon
  2. افتح NVDA → أدوات → إدارة الإضافات
  3. انقر على "تثبيت..." وحدد الملف
  4. أعد تشغيل NVDA عند الطلب

الاستخدام

  1. اضغط NVDA+Ctrl+Shift+F لفتح Padronix
  2. حمّل ملف TXT يحتوي على نص المقال
  3. استورد ملف JSON منظم (اختياري)
  4. انقر على "إنشاء مستند DOCX"
  5. سيتم حفظ الملف في مجلد المستندات

الإعدادات

الوصول: NVDA → تفضيلات → إعدادات → Padronix

  • مزود الذكاء الاصطناعي (Google، OpenAI، OpenRouter، Ollama)
  • مفتاح API
  • النموذج
  • المعيار الأكاديمي (ABNT، APA، IEEE، إلخ)

المتطلبات

  • NVDA 2024.1 أو أعلى
  • Python 3.11+ (مضمن في NVDA 2026+)
  • اتصال بالإنترنت (باستثناء Ollama المحلي)
  • المكتبات: python-docx، lxml (مضمنة)

الترخيص

GPL v3+ - رخصة جنو العمومية الإصدار 3 أو أعلى
