Padronix - Akademischer Artikel-Formatierer

Version: 2026.1.0

Beschreibung

Padronix ist eine NVDA-Erweiterung, die bei der Strukturierung und Formatierung akademischer Artikel hilft. Mit der Integration von KI-Modellen ermöglicht es die Erstellung von Dokumenten nach den Standards ABNT, APA, IEEE und anderen akademischen Formaten auf zugängliche und automatisierte Weise.

Installation

  1. Laden Sie die .nvda-addon-Datei herunter
  2. Öffnen Sie NVDA → Werkzeuge → Erweiterungen verwalten
  3. Klicken Sie auf "Installieren..." und wählen Sie die Datei aus
  4. Starten Sie NVDA bei Aufforderung neu

Verwendung

  1. Drücken Sie NVDA+Ctrl+Shift+F, um Padronix zu öffnen
  2. Laden Sie eine TXT-Datei mit dem Artikeltext
  3. Importieren Sie eine strukturierte JSON-Datei (optional)
  4. Klicken Sie auf "DOCX-Dokument generieren"
  5. Die Datei wird im Ordner Dokumente gespeichert

Einstellungen

Zugriff: NVDA → Einstellungen → Einstellungen → Padronix

  • KI-Anbieter (Google, OpenAI, OpenRouter, Ollama)
  • API-Schlüssel
  • Modell
  • Akademischer Standard (ABNT, APA, IEEE, etc.)

Anforderungen

  • NVDA 2024.1 oder höher
  • Python 3.11+ (in NVDA 2026+ enthalten)
  • Internetverbindung (außer für lokales Ollama)
  • Bibliotheken: python-docx, lxml (enthalten)
