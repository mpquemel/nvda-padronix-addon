Padronix - Formattatore di Articoli Accademici

Versione: 2026.1.0

Descrizione

Padronix è un componente aggiuntivo per NVDA che aiuta a strutturare e formattare articoli accademici. Con l'integrazione di modelli IA, consente di generare documenti secondo gli standard ABNT, APA, IEEE e altri formati accademici in modo accessibile e automatizzato.

Installazione

  1. Scarica il file .nvda-addon
  2. Apri NVDA → Strumenti → Gestione componenti aggiuntivi
  3. Clicca su "Installa..." e seleziona il file
  4. Riavvia NVDA quando richiesto

Utilizzo

  1. Premi NVDA+Ctrl+Shift+F per aprire Padronix
  2. Carica un file TXT con il testo dell'articolo
  3. Importa un file JSON strutturato (opzionale)
  4. Clicca su "Genera documento DOCX"
  5. Il file verrà salvato nella cartella Documenti

Impostazioni

Accesso: NVDA → Preferenze → Impostazioni → Padronix

  • Fornitore IA (Google, OpenAI, OpenRouter, Ollama)
  • Chiave API
  • Modello
  • Standard accademico (ABNT, APA, IEEE, ecc.)

Requisiti

  • NVDA 2024.1 o superiore
  • Python 3.11+ (incluso in NVDA 2026+)
  • Connessione Internet (tranne per Ollama locale)
  • Librerie: python-docx, lxml (incluse)
