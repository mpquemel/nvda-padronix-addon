Padronix - Estruturador de Artigos Acadêmicos

Versão: 2026.1.0

Descrição

Padronix é um complemento para NVDA que auxilia na estruturação e formatação de artigos acadêmicos. Com integração a modelos de IA, permite gerar documentos no padrão ABNT, APA, IEEE e outros formatos acadêmicos de forma acessível e automatizada.

Instalação

  1. Baixe o arquivo .nvda-addon
  2. Abra NVDA → Ferramentas → Gerenciar complementos
  3. Clique em "Instalar..." e selecione o arquivo
  4. Reinicie o NVDA quando solicitado

Uso

  1. Pressione NVDA+Ctrl+Shift+F para abrir o Padronix
  2. Carregue um arquivo TXT com o texto do artigo
  3. Importe um JSON estruturado (opcional)
  4. Clique em "Gerar Documento DOCX"
  5. O arquivo será salvo na pasta Documentos

Configurações

Acesse: NVDA → Preferências → Configurações → Padronix

  • Provedor de IA (Google, OpenAI, OpenRouter, Ollama)
  • Chave API
  • Modelo
  • Padrão acadêmico (ABNT, APA, IEEE, etc.)

Requisitos

  • NVDA 2024.1 ou superior
  • Python 3.11+ (incluído no NVDA 2026+)
  • Conexão com internet (exceto Ollama local)
  • Bibliotecas: python-docx, lxml (incluídas)
