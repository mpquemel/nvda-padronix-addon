Padronix - Academic Article Formatter

Version: 2026.1.0

Description

Padronix is an NVDA add-on that assists in structuring and formatting academic articles. With AI model integration, it enables generating documents in ABNT, APA, IEEE, and other academic formats in an accessible and automated manner.

Installation

  1. Download the .nvda-addon file
  2. Open NVDA → Tools → Manage Add-ons
  3. Click "Install..." and select the file
  4. Restart NVDA when prompted

Usage

  1. Press NVDA+Ctrl+Shift+F to open Padronix
  2. Load a TXT file with the article text
  3. Import a structured JSON file (optional)
  4. Click "Generate DOCX Document"
  5. The file will be saved to the Documents folder

Settings

Access: NVDA → Preferences → Settings → Padronix

  • AI Provider (Google, OpenAI, OpenRouter, Ollama)
  • API Key
  • Model
  • Academic Standard (ABNT, APA, IEEE, etc.)

Requirements

  • NVDA 2024.1 or higher
  • Python 3.11+ (included in NVDA 2026+)
  • Internet connection (except for local Ollama)
  • Libraries: python-docx, lxml (included)
