Padronix - Formateador de Artículos Académicos

Versión: 2026.1.0

Descripción

Padronix es un complemento para NVDA que ayuda a estructurar y formatear artículos académicos. Con integración a modelos de IA, permite generar documentos en formato ABNT, APA, IEEE y otros estándares académicos de forma accesible y automatizada.

Instalación

  1. Descargue el archivo .nvda-addon
  2. Abra NVDA → Herramientas → Administrar complementos
  3. Haga clic en "Instalar..." y seleccione el archivo
  4. Reinicie NVDA cuando se le solicite

Uso

  1. Presione NVDA+Ctrl+Shift+F para abrir Padronix
  2. Cargue un archivo TXT con el texto del artículo
  3. Importe un archivo JSON estructurado (opcional)
  4. Haga clic en "Generar Documento DOCX"
  5. El archivo se guardará en la carpeta Documentos

Configuración

Acceso: NVDA → Preferencias → Configuración → Padronix

  • Proveedor de IA (Google, OpenAI, OpenRouter, Ollama)
  • Clave API
  • Modelo
  • Estándar académico (ABNT, APA, IEEE, etc.)

Requisitos

  • NVDA 2024.1 o superior
  • Python 3.11+ (incluido en NVDA 2026+)
  • Conexión a Internet (excepto para Ollama local)
  • Bibliotecas: python-docx, lxml (incluidas)
