1. Sobre Padronix

Padronix es un complemento para NVDA que estructura y formatea artículos académicos automáticamente usando Inteligencia Artificial.

Características principales:
  • Estructuración automática de artículos vía IA
  • 7 estándares académicos: ABNT, APA, IEEE, Chicago, Harvard, MLA, Vancouver
  • 4 proveedores de IA: Google Gemini, OpenAI, OpenRouter, Ollama Local
  • Interfaz 100% accesible con NVDA
  • Retroalimentación auditiva durante todo el procesamiento
  • Soporte para 7 idiomas

2. Instalación

2.1 Pré-requisitos

  • NVDA: Versión 2024.1 o superior
  • Windows: 10 u 11
  • Python: 3.11+ (incluido con NVDA)

2.2 Pasos para Instalar

  1. Descargue el archivo padronix-2026.1.0.nvda-addon
  2. Con NVDA abierto, presione NVDA+N para abrir el menú
  3. Navegue a Herramientas → Administrar Complementos
  4. Presione I para Instalar
  5. Seleccione el archivo .nvda-addon descargado
  6. Confirme la instalación
  7. Reinicie NVDA cuando se solicite (presione NVDA+Q, luego Enter)
✅ Verificación: Después de reiniciar, presione NVDA+Ctrl+Shift+F. Debe escuchar: "Padronix abierto".

3. Configuración de Proveedores de IA

3.1 Google Gemini (Recomendado para contexto largo)

Cómo obtener la clave API de Google:

  1. Visite Google AI Studio
  2. Inicie sesión con su cuenta de Google
  3. Haga clic en "Create API Key"
  4. Seleccione o cree un proyecto
  5. Copie la clave generada (comienza con AIza...)
  6. ¡Guárdela en un lugar seguro!

3.2 OpenAI (GPT-4, GPT-3.5)

Cómo obtener la clave API de OpenAI:

  1. Visite OpenAI Platform
  2. Inicie sesión o cree una cuenta
  3. Haga clic en "+ Create new secret key"
  4. Copie la clave inmediatamente (comienza con sk-proj-...)

3.3 OpenRouter (Múltiples modelos)

Cómo obtener la clave API de OpenRouter:

  1. Visite OpenRouter Keys
  2. Inicie sesión
  3. Haga clic en "Create Key"
  4. Copie la clave (comienza con sk-or-...)

3.4 Ollama Local (100% offline y gratuito)

Paso 1: Descargar e instalar Ollama

  1. Visite ollama.ai/download
  2. Haga clic en "Download for Windows"
  3. Ejecute OllamaSetup.exe
  4. Siga las instrucciones de instalación

Paso 2: Descargar un modelo

ollama pull llama3

4. Cómo Usar

4.1 Abrir Padronix

Presione: NVDA+Ctrl+Shift+F

4.2 Flujo de Trabajo

  1. Presione Alt+C para cargar TXT
  2. Seleccione el estándar académico
  3. Presione Alt+G para generar DOCX
  4. Espere el sonido de éxito
