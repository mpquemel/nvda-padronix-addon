1. Informazioni su Padronix

Padronix è un add-on NVDA che struttura e formatta automaticamente articoli accademici con l'IA.

Caratteristiche:
  • 7 standard: ABNT, APA, IEEE, Chicago, Harvard, MLA, Vancouver
  • 4 fornitori IA: Google Gemini, OpenAI, OpenRouter, Ollama
  • Interfaccia 100% accessibile
  • Supporto 7 lingue

2. Installazione

Prerequisiti

  • NVDA 2024.1+
  • Windows 10/11
  • Python 3.11+

Passaggi

  1. Scaricare padronix-2026.1.0.nvda-addon
  2. Premere NVDA+N → Strumenti → Gestione componenti aggiuntivi
  3. Premere I per Installare
  4. Selezionare il file scaricato
  5. Riavviare NVDA

3. Configurazione IA

Google Gemini

  1. Visitare Google AI Studio
  2. Creare chiave API
  3. Copiare (AIza...)

OpenAI

  1. Visitare OpenAI Platform
  2. Creare nuova chiave segreta
  3. Copiare (sk-proj-...)

OpenRouter

  1. Visitare OpenRouter
  2. Creare chiave (sk-or-...)

Ollama Locale (100% offline)

  1. Scaricare ollama.ai
  2. Eseguire: 
    ollama pull llama3
  3. Verificare: 
    ollama list

4. Come Utilizzare

Aprire: NVDA+Ctrl+Shift+F

Flusso di lavoro

  1. Alt+C - Carica TXT
  2. Seleziona standard accademico
  3. Alt+G - Genera DOCX
  4. Aspetta il suono di successo
