1. Über Padronix

Padronix ist ein NVDA-Add-on, das akademische Artikel automatisch mit KI strukturiert und formatiert.

Funktionen:
  • 7 Standards: ABNT, APA, IEEE, Chicago, Harvard, MLA, Vancouver
  • 4 KI-Anbieter: Google Gemini, OpenAI, OpenRouter, Ollama
  • 100% barrierefreie Oberfläche
  • 7 Sprachen unterstützt

2. Installation

Voraussetzungen

  • NVDA 2024.1+
  • Windows 10/11
  • Python 3.11+

Schritte

  1. Download padronix-2026.1.0.nvda-addon
  2. Drücken NVDA+N → Werkzeuge → Add-ons verwalten
  3. Drücken I für Installieren
  4. Datei auswählen
  5. NVDA neu starten

3. KI-Konfiguration

Google Gemini

  1. Besuchen Google AI Studio
  2. API-Schlüssel erstellen
  3. Kopieren (AIza...)

OpenAI

  1. Besuchen OpenAI Platform
  2. Neuen geheimen Schlüssel erstellen
  3. Kopieren (sk-proj-...)

OpenRouter

  1. Besuchen OpenRouter
  2. Schlüssel erstellen (sk-or-...)

Ollama Lokal (100% offline)

  1. Download ollama.ai
  2. Ausführen: 
    ollama pull llama3
  3. Überprüfen: 
    ollama list

4. Verwendung

Öffnen: NVDA+Ctrl+Shift+F

Workflow

  1. Alt+C - TXT laden
  2. Akademischen Standard wählen
  3. Alt+G - DOCX generieren
  4. Erfolgston abwarten
