1. حول بادرونكس

بادرونكس هو إضافة NVDA تقوم بهيكلة وتنسيق المقالات الأكاديمية تلقائياً باستخدام الذكاء الاصطناعي.

الميزات الرئيسية:
  • 7 معايير: ABNT، APA، IEEE، Chicago، Harvard، MLA، Vancouver
  • 4 مزودي ذكاء اصطناعي: Google Gemini، OpenAI، OpenRouter، Ollama
  • واجهة 100% متاحة
  • دعم 7 لغات

2. التثبيت

المتطلبات المسبقة

  • NVDA 2024.1+
  • Windows 10/11
  • Python 3.11+

الخطوات

  1. تنزيل padronix-2026.1.0.nvda-addon
  2. اضغط NVDA+N → أدوات → إدارة الإضافات
  3. اضغط I للتثبيت
  4. حدد الملف المحمل
  5. أعد تشغيل NVDA

3. إعداد الذكاء الاصطناعي

Google Gemini

  1. زيارة Google AI Studio
  2. إنشاء مفتاح API
  3. نسخ (AIza...)

OpenAI

  1. زيارة OpenAI Platform
  2. إنشاء مفتاح سري جديد
  3. نسخ (sk-proj-...)

OpenRouter

  1. زيارة OpenRouter
  2. إنشاء مفتاح (sk-or-...)

Ollama المحلي (100% غير متصل)

  1. تنزيل ollama.ai
  2. تشغيل: 
    ollama pull llama3
  3. التحقق: 
    ollama list

4. كيفية الاستخدام

الفتح: NVDA+Ctrl+Shift+F

سير العمل

  1. Alt+C - تحميل TXT
  2. اختيار المعيار الأكاديمي
  3. Alt+G - إنشاء DOCX
  4. انتظار صوت النجاح
